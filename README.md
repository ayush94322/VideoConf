# VideoConf – Real-Time Video Calling & Chat Application

VideoConf is a full-stack real-time video conferencing and live chat web application inspired by Zoom. It enables users to connect instantly through video calls and messaging.

«Note: This project is currently optimized for desktop. Responsive design for mobile devices is under development.»

---

## Features

- Real-time video calling
- Live chat during meetings
- Multiple participants support
- Secure authentication (JWT)
- Peer-to-peer connection using WebRTC
- Unique room ID for joining calls
- Screen sharing feature
- Mute/unmute & video controls

---

## Tech Stack

### Frontend:

- React.js
- Material UI (MUI)
- HTML5, CSS3, JavaScript

### Backend:

- Node.js
- Express.js

### Database:

- MongoDB

### Real-Time Communication:

- WebRTC
- Socket.io
